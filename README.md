# Nativify

This repository is the public issue tracker for Nativify's web service, API, dashboard, and CI integrations. The Nativify source code is maintained separately.

## Report an issue

[Open an issue](https://github.com/neilhuang007/nativify/issues/new) after checking for an existing report. Include the affected component, reproduction steps, expected and actual behavior, and relevant redacted logs. Never post API keys, credentials, proprietary JARs, or other secrets.

Documentation: https://nativify.javavirtualenvironment.com/docs/
